# fk_affiliate

Flutter library to fetch Deals of the day and Offers from Flipkart Affiliate API

## Getting Started
```
dependencies:
    fk_affiliate:
    git:
      url: https://github.com/sanjeevbishnoi/flutter-flipkart-affiliate
      ref: main
```
